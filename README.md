# ML-Based Student Mental Health Prediction

## Overview

This project applies Machine Learning techniques to predict student mental health conditions, specifically **Depression**, **Anxiety**, and **Stress**, using the **Depression Data Dataset**. The project includes data preprocessing, exploratory data analysis, feature engineering, supervised learning, regression analysis, and unsupervised learning to identify patterns affecting students' mental well-being.

## Objectives

* Analyze factors associated with student mental health.
* Predict Depression, Anxiety, and Stress levels using Machine Learning models.
* Compare multiple classification and regression algorithms.
* Evaluate model performance using standard metrics.
* Explore hidden patterns through clustering techniques.

## Dataset

The project uses the **Depression Data Dataset**, containing demographic, academic, lifestyle, and mental health-related attributes.

### Example Features

* Age
* Gender
* Academic Year
* CGPA
* Scholarship Status
* Study Satisfaction
* Sleep Duration
* Dietary Habits
* Academic Pressure
* Financial Stress
* Study Hours
* Social Relationships

### Target Variables

#### Classification Targets

* Anxiety Label
* Stress Label
* Depression Label

#### Regression Targets

* Anxiety Score
* Stress Score
* Depression Score
